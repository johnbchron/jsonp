# jsonp

This is just a simple JSON prettifier.
Hopefully it's helpful to you.

## Contributing

Thanks so much for considering contributing!
Here's generally the technologies we use:

- `direnv` - for automatically loading the environment
- `nix` - for packaging the project and building the environment
- `rust` - for the actual code
- `leptos` - for the web stuff

If you have an improvement you'd like to make, please don't hesitate to file a PR.
